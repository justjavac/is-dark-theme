## Installation

Install the package using `npm` or `yarn`.

**use npm**:

```bash
npm install is-dark-theme
```

or **use yarn**:

```bash
yarn add is-dark-theme
```

## Usage

```js
import isDarkTheme from 'is-dark-theme'

isDarkTheme() // true | false
```
